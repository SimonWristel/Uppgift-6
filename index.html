HTML

STRUKTUR

<!DOCTYPE>

Denna deklaration specifierar vilken verision av HTML som används i dokumentet.

<html>

Rotelementet i ett HTML-dokument som innehåller alla andra element.

<head>

En behållare för metadata om dokumentet, såsom titeln, länkar till stilmallar och metataggar.

<title>

Definierar titeln på HTML-dokumentet, som visas i webbläsarens titelrad eller flik.

<style>

Används för att bädda in CSS (Cascading Style Sheets) i ett HTML-dokument.

<link>

Specifierar externa resurser, som CSS-filer, att användas i dokument.

<body>

Huvudinnehållsområdet i HTML-dokumentet där text, bilder och annat innehåll visas.

SEMANTISKA ELEMENT

<main>

Representerar Huvudinnehållsområdet i HTML-dokumentet, och det bör endast finnas ett <main>-element i en HTML-sida.

<header>

Innehåller vanligtvis introducerande innehåll eller en webbplatslogotyp.
